# 语文字词表上传指南

## Markdown 格式示例

### 基础格式

```markdown
# 七年级上册字词表

## 第一单元

### 词语
- 侍弄（shì nòng）：照料；喂养
- 和蔼（hé ǎi）：态度温和，容易接近
- 骊歌（lí gē）：告别的歌
- 徜徉（cháng yáng）：闲游；安闲自在地步行
- 训诫（xùn jiè）：教导，告诫
- 花团锦簇（huā tuán jǐn cù）：形容色彩缤纷

### 生字
- 髫（tiáo）：小孩子垂下的头发
- 髦（máo）：古代称小孩子垂下来的头发

## 第二单元

### 词语
- 乃至（nǎi zhì）：甚至
- 脊梁（jǐ liáng）：脊背
- 傲然（ào rán）：坚强不屈的样子
```

### 格式规则

| 项目 | 格式 | 示例 | 必填 |
|------|------|------|------|
| **知识库名称** | `# {名称}` | `# 七年级上册字词表` | ✓ |
| **单元/章节** | `## {名称}` | `## 第一单元` | - |
| **分类** | `### {类型}` | `### 词语` | - |
| **词语条目** | `- {词语}（{拼音}）：{释义}` | `- 侍弄（shì nòng）：照料；喂养` | ✓ |

### 注意事项

1. **拼音格式**：
   - 必须带声调：`shì nòng` ✓
   - 不能用数字：`shi4 nong4` ✗

2. **多个释义**：
   - 使用分号分隔：`照料；喂养`

3. **成语**：
   - 格式相同，无需特殊处理

4. **生字**：
   - 单字也可以用相同格式

## 快速上传（3 种方式）

### 方式一：使用上传界面（推荐，即将推出）

1. 访问 `/knowledge-base` 页面
2. 点击"上传知识库"
3. 选择 Markdown 文件
4. 预览解析结果
5. 确认上传

### 方式二：直接创建文件（快速测试）

创建文件：`data/knowledge-base/chinese/demo/words.json`

```json
{
  "libraryId": "demo",
  "name": "七年级上册字词表",
  "description": "示例字词库",
  "words": [
    {
      "id": "word-1",
      "word": "侍弄",
      "pinyin": "shì nòng",
      "meanings": ["照料", "喂养"],
      "category": "词语",
      "sourceUnit": "第一单元"
    },
    {
      "id": "word-2",
      "word": "和蔼",
      "pinyin": "hé ǎi",
      "meanings": ["态度温和，容易接近"],
      "category": "词语",
      "sourceUnit": "第一单元"
    }
  ]
}
```

### 方式三：使用 API 上传

```bash
curl -X POST http://localhost:3001/api/knowledge-base/upload \
  -H "Content-Type: application/json" \
  -d '{
    "subject": "chinese",
    "name": "七年级上册字词表",
    "markdown": "# 七年级上册字词表\n\n## 第一单元\n\n### 词语\n\n- 侍弄（shì nòng）：照料；喂养\n- 和蔼（hé ǎi）：态度温和，容易接近"
  }'
```

### 填写示例

```markdown
# 七年级上册字词表

## 第一单元：从百草园到三味书屋

### 词语
- 侍弄（shì nòng）：照料；喂养
- 和蔼（hé ǎi）：态度温和，容易接近
- 骊歌（lí gē）：告别的歌
- 徜徉（cháng yáng）：闲游；安闲自在地步行
- 训诫（xùn jiè）：教导，告诫
- 肥硕（féi shuò）：又大又饱满

### 生字
- 髫（tiáo）：小孩子垂下的头发
- 髦（máo）：古代称小孩子垂下来的头发
- 玫（méi）：一种美玉
- 珊（shān）：珊瑚

## 第二单元：成长的脚步

### 词语
- 乃至（nǎi zhì）：甚至
- 脊梁（jǐ liáng）：脊背
- 傲然（ào rán）：坚强不屈的样子
- 纯粹（chún cuì）：不掺杂别的成分
- 闲适（xián shì）：清闲安逸

### 成语
- 花团锦簇（huā tuán jǐn cù）：形容色彩缤纷
- 美不胜收（měi bù shèng shōu）：美好的东西太多，看不过来
```

## 常见问题

### Q: 如何获取教材字词表？

A: 有以下方式：
1. 从教材附录扫描
2. 从教师用书抄录
3. 从在线资源下载（如教材官网、教学资源网站）

### Q: 支持批量导入吗？

A: 暂不支持，但您可以把多个字词表合并在一个 Markdown 文件中。

### Q: 如何编辑已上传的字词表？

A: 计划中的功能，目前可以先删除后重新上传。

### Q: 支持导出吗？

A: 计划支持导出为 Markdown 或 Excel 格式。
