## ✍️ 考试流程

### 考试界面

```
┌─────────────────────────────────────┐
│  字词考试 - 第 1 / 10 题               │
├─────────────────────────────────────┤
│                                             │
│  拼音：yuán xūn                           │
│  释义：立大功的人                        │
│                                             │
│  ┌─────────────────────────────────┐  │
│  │  请手写该字词：                  │  │
│  │  [手写画板 - 500x250]             │  │
│  │                                     │  │
│  └─────────────────────────────────┘  │
│                                             │
│  [退出]              [重写] [提交答案]   │
│                                             │
└─────────────────────────────────────┘
```

### 识别和判题

1. **手写字词**
   - 在画板上用鼠标或触控笔书写

2. **提交答案**
   - 点击"提交答案"按钮

3. **AI 识别**
   - 系统自动识别手写内容
   - 使用多模态 AI 进行判题
   - 给出详细反馈

4. **查看结果**
   - ✓ 正确：显示绿色对钩
   - ✗ 错误：显示红色叉号，并给出正确答案和提示

### 示例

**题目**：
- 拼音：yuán xūn
- 释义：立大功的人

**用户手写**：元勋

**AI 识别结果**：
```json
{
  "recognizedText": "元勋",
  "isCorrect": true,
  "feedback": "识别正确！书写工整，笔画清晰。"
}
```

**如果写错了**（如写成了"元助"）：
```json
{
  "recognizedText": "元助",
  "isCorrect": false,
  "feedback": "识别结果：元助，与正确答案"元勋"相似66.7%。"勋"字的右边部分需要写得饱满一些。"
}
```

## 💡 使用技巧

### 1. 书写建议

- ✅ **书写工整**：笔画清晰，结构标准
- ✅ **笔顺正确**：按照正确的笔顺书写
- ✅ **大小适中**：字不要太小也不要太大
- ✅ **留白充足**：字与字之间保持适当距离

### 2. 设备选择

- **触控笔**：推荐使用 Apple Pencil 或触控笔
- **鼠标**：也可以使用鼠标书写
- **手机**：支持触摸屏手写

### 3. 考试策略

- **复习模式**：考全部字词，系统复习
- **练习模式**：随机10题，日常练习
- **针对性训练**：记录错题，重点练习
