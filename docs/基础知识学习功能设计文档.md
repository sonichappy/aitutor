## 1. 需求概述

### 1.1 背景

当前 AI 导师系统主要针对试卷识别和分析，但对于语文学科的**字词积累**和英语学科的**词汇记忆**等基础知识学习缺乏系统化支持。本功能旨在：

1. 建立可管理的**基础知识库**（语文字词、英文单词）
2. 提供**在线考核**功能，生成基础知识试卷
3. 支持**手写识别**，满足语文手写考核需求
4. 记录学习数据，**纳入个性化学习分析**

### 1.2 目标用户

- **学生**：通过日常积累和考核提升基础词汇能力
- **家长/老师**：管理和监督学生的基础知识学习进度

### 1.3 适用学科

| 学科 | 知识类型 | 考核方式 |
|------|---------|---------|
| **语文** | 字词（拼音、释义、组词） | 选择题 + 手写题 |
| **英语** | 单词（拼写、释义、例句） | 选择题 + 拼写题 |

### 1.4 核心功能


#### 2.1.1 上传字词/单词表

**输入格式**：Markdown 文件

```markdown
- 侍**弄**（shì nòng）：照料；喂养
- **和**蔼（hé ǎi）：态度温和，容易接近
- **骊**歌（lí gē）：告别的歌
- **徜**徉（cháng yáng）：闲游；安闲自在地步行

### 生字
```

### 2.2 试卷生成

#### 2.2.1 生成流程

```
1. 用户选择知识库和考核范围
   ↓
2. 配置试卷参数
   - 题目数量
   - 题型分布（选择题、手写题、拼写题）
   - 难度级别
   ↓
3. 系统随机抽取词汇
   ↓
4. 生成不同题型
   - 选择题：从释义选择词语/单词
   - 手写题（语文）：看拼音写汉字
   - 拼写题（英语）：看释义写单词
   ↓
5. 生成试卷数据
   ↓
6. 保存到 /data/exams/ 目录
   ↓
7. 跳转到答题页面
```

#### 2.2.2 题型设计

##### 语文题型

| 题型 | 说明 | 示例 |
|------|------|------|
| **拼音选择题** | 给出拼音，选择对应汉字 | 下列词语中，拼音为"cháng yáng"的是？<br/>A. 徜徉  B. 常样  C. 长杨 |
| **释义选择题** | 给出释义，选择对应词语 | "态度温和，容易接近"形容的是哪个词？<br/>A. 和蔼  B. 和气  C. 温和 |
| **手写题** | 给出拼音，手写汉字 | 请写出"shì nòng"对应的词语 |
| **组词选择题** | 给出单字，选择正确组词 | 下列组词正确的是？<br/>A. 侍弄  B. 待弄  C. 是弄 |

##### 英语题型

| 题型 | 说明 | 示例 |
|------|------|------|
| **英译中选择题** | 给出英文单词，选择中文释义 | "apple" 的意思是？<br/>A. 苹果  B. 香蕉  C. 橘子 |
| **中译英选择题** | 给出中文释义，选择英文单词 | "书籍"的英文是？<br/>A. book  B. look  C. cook |
| **拼写题** | 给出中文释义和首字母，拼写单词 | b___ (书籍) |
| **音标题** | 给出单词，选择正确音标 | "book" 的音标是？<br/>A. /bʊk/  B. /buːk/  C. /bɒk/ |

### 2.4 智能判题

#### 2.4.1 判题流程

```
1. 用户提交答案
   ↓
2. 选择题：直接比对选项
   ↓
3. 手写题（语文）：
   - 调用手写识别 API
   - 获取识别文本
   - 与标准答案比对
   - 记录相似度和判题结果
   ↓
4. 拼写题（英语）：
   - 归一化处理（大小写、空格）
   - 精确匹配
   ↓
5. 更新题目状态（isCorrect）
   ↓
6. 如果错误，记录到薄弱知识点
   ↓
7. 更新答题统计
```

#### 2.4.2 手写识别判题规则

```typescript
interface HandwritingResult {
  recognizedText: string      // 识别结果
  confidence: number          // 置信度 0-1
  isCorrect: boolean          // 是否正确
  similarity: number          // 相似度 0-1
  feedback: string            // 反馈信息
}

// 判题逻辑
function judgeHandwriting(
  userAnswer: string,      // 手写识别结果
  correctAnswer: string,   // 正确答案
  confidence: number       // 识别置信度
): HandwritingResult {
  // 1. 完全匹配
  if (userAnswer === correctAnswer) {
    return {
      recognizedText: userAnswer,
      confidence,
      isCorrect: true,
      similarity: 1.0,
      feedback: '完全正确！'
    }
  }

  // 2. 计算编辑距离相似度
  const similarity = calculateLevenshteinSimilarity(userAnswer, correctAnswer)

  // 3. 相似度 >= 0.8 且置信度 >= 0.7，认为正确
  if (similarity >= 0.8 && confidence >= 0.7) {
    return {
      recognizedText: userAnswer,
      confidence,
      isCorrect: true,
      similarity,
      feedback: `识别结果：${userAnswer}，正确答案：${correctAnswer}（相似度${(similarity * 100).toFixed(0)}%，判为正确）`
    }
  }

  // 4. 相似度 >= 0.5，给予部分反馈
  if (similarity >= 0.5) {
    return {
      recognizedText: userAnswer,
      confidence,
      isCorrect: false,
      similarity,
      feedback: `识别结果：${userAnswer}，与正确答案"${correctAnswer}"相似但不同`
    }
  }

  // 5. 完全错误
  return {
    recognizedText: userAnswer,
    confidence,
    isCorrect: false,
    similarity,
    feedback: `识别结果：${userAnswer}，正确答案：${correctAnswer}`
  }
}
```

## 7. 手写识别技术方案

### 7.1 技术选型

#### 方案一：云端 OCR API（推荐）

| 服务商 | API | 优势 | 劣势 |
|--------|-----|------|------|
| **百度 OCR** | 通用文字识别（含手写） | 准确率高，支持手写 | 需要 API Key |
| **腾讯 OCR** | 手写文字识别 | 准确率高，速度快 | 需要 API Key |
| **阿里云 OCR** | 手写文字识别 | 支持多语言 | 需要 API Key |
| **Google Cloud Vision** | Handwriting OCR | 国际化 | 需要翻墙 |

**推荐方案**：使用**百度 OCR** 或 **腾讯 OCR**

**理由**：
- 手写识别准确率高（>90%）
- 响应速度快（<1s）
- 中文支持好
- 价格合理（QPS 限制内免费或低价）

#### 方案二：前端识别库（备选）

| 库 | 优势 | 劣势 |
|----|------|------|
| **Tesseract.js** | 纯前端，无服务调用 | 手写识别效果差 |
| **MyScript** | 专门的手写识别 | 需要商业授权 |

**不推荐**：前端识别库手写效果不佳，不适合生产环境。

### 7.4 手写画板实现

```tsx
// components/HandwritingCanvas.tsx

'use client'

import { useRef, useEffect, useState } from 'react'

interface Point {
  x: number
  y: number
}

interface Stroke {
  points: Point[]
}

export function HandwritingCanvas({
  width = 400,
  height = 200,
  onStrokeChange,
  onClear
}: {
  width?: number
  height?: number
  onStrokeChange?: (strokes: Stroke[]) => void
  onClear?: () => void
}) {
  const canvasRef = useRef<HTMLCanvasElement>(null)
  const [isDrawing, setIsDrawing] = useState(false)
  const [strokes, setStrokes] = useState<Stroke[]>([])
  const currentStroke = useRef<Point[]>([])

  useEffect(() => {
    const canvas = canvasRef.current
    if (!canvas) return

    const ctx = canvas.getContext('2d')
    if (!ctx) return

    ctx.lineWidth = 3
    ctx.lineCap = 'round'
    ctx.lineJoin = 'round'
    ctx.strokeStyle = '#000'
  }, [])

  const getPoint = (e: React.MouseEvent | React.TouchEvent): Point => {
    const canvas = canvasRef.current
    if (!canvas) return { x: 0, y: 0 }

    const rect = canvas.getBoundingClientRect()
    const clientX = 'touches' in e ? e.touches[0].clientX : e.clientX
    const clientY = 'touches' in e ? e.touches[0].clientY : e.clientY

    return {
      x: clientX - rect.left,
      y: clientY - rect.top
    }
  }

  const startDrawing = (e: React.MouseEvent | React.TouchEvent) => {
    e.preventDefault()
    setIsDrawing(true)
    currentStroke.current = [getPoint(e)]
  }

  const draw = (e: React.MouseEvent | React.TouchEvent) => {
    e.preventDefault()
    if (!isDrawing) return

    const canvas = canvasRef.current
    if (!canvas) return

    const ctx = canvas.getContext('2d')
    if (!ctx) return

    const point = getPoint(e)
    currentStroke.current.push(point)

    // 绘制
    const points = currentStroke.current
    if (points.length >= 2) {
      const lastPoint = points[points.length - 2]
      ctx.beginPath()
      ctx.moveTo(lastPoint.x, lastPoint.y)
      ctx.lineTo(point.x, point.y)
      ctx.stroke()
    }
  }

  const stopDrawing = () => {
    if (!isDrawing) return
    setIsDrawing(false)

    if (currentStroke.current.length > 0) {
      const newStrokes = [...strokes, { points: currentStroke.current }]
      setStrokes(newStrokes)
      onStrokeChange?.(newStrokes)
    }
    currentStroke.current = []
  }

  const clearCanvas = () => {
    const canvas = canvasRef.current
    if (!canvas) return

    const ctx = canvas.getContext('2d')
    if (!ctx) return

    ctx.clearRect(0, 0, canvas.width, canvas.height)
    setStrokes([])
    currentStroke.current = []
    onClear?.()
  }

  const getBase64Image = (): string => {
    const canvas = canvasRef.current
    if (!canvas) return ''
    return canvas.toDataURL('image/png')
  }

  return (
    <div className="inline-block">
      <canvas
        ref={canvasRef}
        width={width}
        height={height}
        className="border border-gray-300 rounded bg-white"
        onMouseDown={startDrawing}
        onMouseMove={draw}
        onMouseUp={stopDrawing}
        onMouseLeave={stopDrawing}
        onTouchStart={startDrawing}
        onTouchMove={draw}
        onTouchEnd={stopDrawing}
      />
      <div className="mt-2 flex gap-2">
        <button
          onClick={clearCanvas}
          className="px-4 py-2 bg-gray-200 rounded hover:bg-gray-300"
        >
          清除
        </button>
        <button
          onClick={() => console.log(getBase64Image())}
          className="px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600"
        >
          预览
        </button>
      </div>
    </div>
  )
}
```

### 7.5 判题与反馈

```typescript
// lib/knowledge-exam/judge.ts

import { recognizeHandwriting } from '@/lib/handwriting/baidu-ocr'
import { calculateLevenshteinSimilarity } from '@/lib/utils/string-similarity'

export async function judgeHandwritingQuestion({
  userImage,
  correctAnswer,
  questionId
}: {
  userImage: string
  correctAnswer: string
  questionId: string
}) {
  // 1. 调用 OCR 识别
  const ocrResult = await recognizeHandwriting(userImage)

  if (!ocrResult.words || ocrResult.words.length === 0) {
    return {
      success: false,
      error: '未能识别手写内容，请重新书写'
    }
  }

  // 2. 提取识别文本
  const recognizedText = ocrResult.words.map(w => w.text).join('')
  const confidence = ocrResult.words[0]?.confidence || 0

  // 3. 计算相似度
  const similarity = calculateLevenshteinSimilarity(recognizedText, correctAnswer)

  // 4. 判定
  let isCorrect = false
  let feedback = ''

  if (similarity === 1) {
    isCorrect = true
    feedback = '完全正确！'
  } else if (similarity >= 0.8 && confidence >= 0.7) {
    isCorrect = true
    feedback = `识别结果：${recognizedText}（相似度${(similarity * 100).toFixed(0)}%，判为正确）`
  } else if (similarity >= 0.5) {
    isCorrect = false
    feedback = `识别结果：${recognizedText}，正确答案：${correctAnswer}`
  } else {
    isCorrect = false
    feedback = `识别结果：${recognizedText}，与正确答案差异较大`
  }

  return {
    success: true,
    isCorrect,
    recognizedText,
    confidence,
    similarity,
    feedback
  }
}
```
