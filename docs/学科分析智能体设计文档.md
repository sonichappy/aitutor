## 1. 概述

### 1.1 背景与目标

随着系统中试卷数据的积累，学生和家长需要一个能够综合评估学习状态、提供科学学习指导的智能分析系统。学科分析智能体旨在：

1. **综合分析** - 整合新试卷、历史试卷和历史报告数据
2. **状态研判** - 准确判断当前学习水平和进步情况
3. **趋势预测** - 分析学习发展趋势，识别潜在风险
4. **计划生成** - 制定个性化、可执行的学习计划
5. **重点指导** - 明确指出重点和难点知识点

### 1.2 核心能力

| 能力 | 描述 |
|------|------|
| **趋势分析** | 基于历史数据计算学习进步/退步趋势，量化进步速度 |
| **状态评估** | 多维度评估学习状态（水平等级、效率、强弱领域） |
| **知识诊断** | 智能识别重点知识点和难点知识点，按优先级排序 |
| **计划生成** | 生成短中长期学习计划，包含每日任务分解 |
| **AI 洞察** | 利用 AI 生成个性化评估和建议 |

### 1.3 设计原则

- **数据驱动** - 所有分析基于真实学习数据
- **渐进增强** - 基础分析无需 AI，深度洞察可选使用 AI
- **个性化** - 每个学生的分析结果和计划都是独特的
- **可执行** - 学习计划具体明确，包含时间估算和优先级
- **正向激励** - 强调进步和成就，鼓励学生持续学习

### 4.1 趋势分析模块 (TrendAnalyzer)

**文件**: `lib/learning/trend-analyzer.ts`

#### 4.1.1 功能描述

#### 4.1.2 核心算法

**线性回归分析**

```typescript
// 计算线性回归斜率
function analyzeTrendPattern(dataPoints: TrendDataPoint[]) {
  const n = dataPoints.length
  const xValues = dataPoints.map((_, i) => i)
  const yValues = dataPoints.map(p => p.accuracy)

  // 计算斜率
  const slope = calculateLinearRegressionSlope(xValues, yValues)

  // 判断趋势
  let overallTrend: 'improving' | 'declining' | 'stable'
  if (Math.abs(slope) < 0.02) {
    overallTrend = 'stable'
  } else if (slope > 0) {
    overallTrend = 'improving'
  } else {
    overallTrend = 'declining'
  }

  // 判断强度
  let trendStrength: 'significant' | 'moderate' | 'slight'
  if (Math.abs(slope) >= 0.05) {
    trendStrength = 'significant'
  } else if (Math.abs(slope) >= 0.03) {
    trendStrength = 'moderate'
  } else {
    trendStrength = 'slight'
  }

  return { overallTrend, trendStrength, slope }
}
```

#### 4.2.3 学习效率计算

```typescript
// 学习效率 = 基础分 + 趋势加分 + 完成率加分 - 重复错误扣分
function calculateLearningEfficiency(
  exams: ExamData[],
  trend?: LearningTrend
): number {
  let efficiency = 50  // 基础分

  // 1. 趋势加分/扣分
  if (trend?.analysis?.overallTrend === 'improving') {
    efficiency += 20
  } else if (trend?.analysis?.overallTrend === 'declining') {
    efficiency -= 10
  }

  // 2. 题目完成率加分
  const completionRate = calculateCompletionRate(exams)
  efficiency += completionRate * 20

  // 3. 重复错误扣分
  const repeatErrorRate = calculateRepeatErrorRate(exams)
  efficiency -= repeatErrorRate * 30

  return Math.max(0, Math.min(100, efficiency))
}
```

#### 4.3.2 优先级计算

```typescript
// 优先级 = 重要性权重 + (100 - 掌握度) * 0.5 + 频率权重
function calculatePriority(
  importance: 'critical' | 'important' | 'normal',
  masteryLevel: number,
  count: number
): number {
  let priority = 0

  // 重要性权重
  const importanceWeight = {
    critical: 50,
    important: 30,
    normal: 10
  }
  priority += importanceWeight[importance]

  // 掌握程度权重（掌握越低越优先）
  priority += (100 - masteryLevel) * 0.5

  // 出现频率权重
  priority += Math.min(count * 5, 25)

  return priority
}
```

#### 4.3.3 学习顺序生成

```typescript
// 建议学习顺序：高频率难点 -> 重要未掌握 -> 次要难点
function generateStudyOrder(
  keyPoints: KeyPoint[],
  difficultPoints: DifficultPoint[]
): string[] {
  const order: string[] = []

  // 1. 高错误率的极难点
  for (const p of difficultPoints) {
    if (p.difficulty === 'very_hard') {
      order.push(p.name)
    }
  }

  // 2. 重要但掌握不好的知识点
  for (const p of keyPoints) {
    if (p.importance === 'critical' && p.recentPerformance === 'poor') {
      order.push(p.name)
    }
  }

  // 3. 其他难点
  for (const p of difficultPoints) {
    if (!order.includes(p.name)) {
      order.push(p.name)
    }
  }

  return order
}
```

#### 4.4.2 短期计划生成

function generateShortTermPlan(
  keyPoints: KeyDifficultPoints,
  state: LearningStateAssessment
) {
  const items: StudyPlanItem[] = []

  // 为每个薄弱点生成复习任务
  keyPoints.difficultPoints.slice(0, 5).forEach((point, index) => {
    items.push({
      id: `st-review-${index}`,
      type: 'review_wrong',
      title: `复习错题：${point.name}`,
      description: `重点复习${point.name}相关错题，错误率${point.errorRate}%`,
      targetKnowledgePoints: [point.name],
      estimatedTime: 30,
      priority: point.difficulty === 'very_hard' ? 'high' : 'medium',
      status: 'pending'
    })
  })

  // 添加每日练习任务
  items.push({
    id: 'st-daily-practice',
    type: 'daily_task',
    title: '每日练习',
    description: '完成当日布置的练习题，巩固所学知识',
    estimatedTime: 45,
    priority: 'high',
    status: 'pending'
  })

  return {
    description: '未来一周重点复习薄弱知识点，完成错题订正',
    items,
    totalEstimatedTime: items.reduce((sum, item) => sum + item.estimatedTime, 0)
  }
}
