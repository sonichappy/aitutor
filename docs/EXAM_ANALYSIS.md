# 试卷分析功能设计

## 功能概述

用户上传/输入考试卷内容，AI 自动分析试卷，提供详细的诊断报告。

---

## 核心功能

### 1. 试卷录入
- **文本粘贴** - 直接粘贴试卷文本
- **图片上传** - 上传试卷图片，OCR 识别
- **文件上传** - 支持 PDF/Word（后续）

### 2. 试卷解析
- AI 识别题目类型（选择、填空、解答）
- 提取题目内容
- 识别知识点
- 确定题目难度

### 3. 答案录入
- 用户填写/选择答案
- 标注作答情况（正确/错误/未做）

### 4. 试卷分析
- 计算得分、正确率
- 知识点掌握度分析
- 错题原因分析
- 薄弱点识别

### 5. 分析报告
- 总体成绩评估
- 知识点分布图
- 错题解析
- 改进建议
- 推荐练习

## 数据模型

### Exam (试卷)
```typescript
{
  id: string
  userId: string
  subject: string          // 科目
  examType: string         // 试卷类型: 期中/期末/月考/练习
  examDate: Date
  totalScore: number       // 总分
  obtainedScore?: number   // 得分
  status: string           // parsing/done/analyzing/completed
  content: string          // 原始内容
  createdAt: Date
}
```

### ExamQuestion (试卷题目)
```typescript
{
  id: string
  examId: string
  questionNumber: number   // 题号
  type: string            // choice/fill/answer
  content: string         // 题目内容
  options?: string[]      // 选项
  knowledgePoints: string[]// 知识点
  difficulty: number      // 1-5
  score: number           // 分值
  correctAnswer?: string  // 正确答案（AI推断或用户标注）
  userAnswer?: string     // 用户答案
  isCorrect?: boolean     // 是否正确
  isSkipped?: boolean     // 是否未做
}
```

### ExamAnalysis (分析结果)
```typescript
{
  id: string
  examId: string
  summary: {
    totalScore: number
    obtainedScore: number
    correctRate: number
    rank?: string
  }
  knowledgeAnalysis: {
    point: string
    correctRate: number
    questions: number
  }[]
  weakPoints: string[]
  strongPoints: string[]
  suggestions: string[]
  aiReport?: string
}
```

## 用户流程

### 方案 A: 完整流程
```
1. 上传试卷（图片/文本）
2. AI 解析题目
3. 用户确认/修正题目
4. 用户录入答案
5. AI 标注正确答案
6. 生成分析报告
```

### 方案 B: 快速流程
```
1. 粘贴试卷文本
2. 同时标注答案和得分
3. AI 一次性分析
```

## API 设计

### POST /api/exam/upload
上传/解析试卷

```json
{
  "content": "试卷文本内容",
  "subject": "数学",
  "examType": "期末"
}
```

### POST /api/exam/:examId/answer
提交答案

```json
{
  "answers": [
    { "questionId": "q1", "userAnswer": "A", "isCorrect": true },
    { "questionId": "q2", "userAnswer": "C", "isCorrect": false }
  ]
}
```

### POST /api/exam/:examId/analyze
生成分析报告

### GET /api/exam/list
获取试卷列表

### GET /api/exam/:id
获取试卷详情
