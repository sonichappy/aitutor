# AITutor UI/UX 设计文档

## 1. 设计理念

### 核心原则
- **简洁清晰** - 界面简洁，功能明确，减少认知负担
- **激励导向** - 通过进度、成就等元素激发学习动力
- **个性化** - 基于学生水平提供定制化体验
- **友好亲和** - 色彩柔和，交互友好，营造轻松学习氛围

### 品牌色彩
```
主色调:   蓝色 #3B82F6 (学习、专业、信任)
辅助色:   绿色 #10B981 (进步、正确)
警告色:   红色 #EF4444 (错误、薄弱)
中性色:   灰色 #6B7280 (文本、边框)
背景色:   浅灰 #F9FAFB (页面背景)
```
