## 🚀 快速开始

### 步骤 1：安装依赖

```bash
# 安装 axios（用于 HTTP 请求）
npm install axios
# 或
pnpm install axios
```

### 步骤 2：配置 OCR 服务

#### 腾讯云（推荐）

```bash
# 复制配置模板
cp .env.example.ocr .env.local

# 编辑 .env.local
OCR_PROVIDER=tencent
TENCENT_OCR_SECRET_ID=你的SecretId
TENCENT_OCR_SECRET_KEY=你的SecretKey
```

#### 百度云

```bash
OCR_PROVIDER=baidu
BAIDU_OCR_API_KEY=你的API_Key
BAIDU_OCR_SECRET_KEY=你的Secret_Key
```

### 步骤 3：启动项目

```bash
npm run dev
# 或
pnpm dev
```

### 步骤 4：访问演示页面

```
http://localhost:3000/demo/handwriting
```

## 💡 使用示例

### 基础用法

```tsx
'use client'

import { HandwritingInput } from '@/components/HandwritingInput'

export default function Page() {
  return (
    <HandwritingInput
      width={400}
      height={200}
      correctAnswer="侍弄"
      onRecognized={({ text, confidence }) => {
        console.log('识别结果:', text)
      }}
      onJudged={(result) => {
        console.log('判题结果:', result.isCorrect)
      }}
    />
  )
}
```

### API 调用

```tsx
// 识别
const response = await fetch('/api/handwriting/recognize', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ image: 'data:image/png;base64,...' })
})
const data = await response.json()
console.log(data.data.text)

// 判题
const response2 = await fetch('/api/handwriting/judge', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    image: 'data:image/png;base64,...',
    correctAnswer: '侍弄'
  })
})
const data2 = await response2.json()
console.log(data2.data.isCorrect)
```

## 🔧 配置选项

### 判题配置

```tsx
<HandwritingInput
  correctAnswer="侍弄"
  options={{
    similarityThreshold: 0.8, // 相似度阈值（0-1）
    confidenceThreshold: 0.7, // 置信度阈值（0-1）
    lenientMode: true, // 宽松模式
    provider: 'tencent' // 指定 OCR 服务商
  }}
/>
```

### 画板配置

```tsx
<HandwritingCanvas
  width={400} // 宽度
  height={200} // 高度
  strokeWidth={3} // 笔画粗细
  strokeColor="#000000" // 笔画颜色
  backgroundColor="#ffffff" // 背景颜色
  enableTouch={true} // 启用触摸
  placeholder="请在此处手写..." // 占位符
/>
```

## 🆘 常见问题

### Q: 如何切换 OCR 服务商？

A: 修改 `.env.local` 文件中的 `OCR_PROVIDER` 变量。

### Q: 识别失败怎么办？

A:
1. 检查 API 密钥配置
2. 检查网络连接
3. 查看服务端日志
4. 确认 API 额度是否用尽

### Q: 如何提高识别准确率？

A:
1. 提示用户书写工整
2. 使用足够大的画布（建议 400x200+）
3. 启用宽松模式
4. 提供重写功能

### Q: 支持其他语言吗？

A: 腾讯云和百度云都支持多语言，具体请参考官方文档。
